# Tokenizar_Poema
Tokenizar un poema en Python es básicamente agarrar el texto completo y partirlo en pedazos más chicos para poder analizarlo, ya sea por palabras o por cada línea (verso). Es el primer paso si quieres que la computadora "entienda" la estructura de lo que escribiste.
